# TP3 — Analyse avancée : Window Functions et indicateurs boursiers
### BigData2 — Niveau : Avancé

---

## Contexte

Vous disposez de l'historique complet de l'action **Apple (AAPL)** depuis 1980 (`data/AAPL.csv`), soit plus de **8700 jours de cotation**.

Dans ce TP, vous allez calculer des **indicateurs financiers réels** utilisés par les traders :
- La **moyenne mobile** (tendance sur N jours)
- La **variation en pourcentage** d'une journée à l'autre
- Le **classement** des meilleures et pires journées
- La **fenêtre glissante** pour identifier les tendances

Ces calculs nécessitent les **Window Functions** de Spark SQL — des fonctions qui opèrent sur un groupe de lignes en tenant compte de leur ordre.

---

## Rappel : Window Functions Spark SQL

Une *window function* s'applique sur une **partition** de données ordonnées :

```sql
-- Syntaxe générale
FONCTION() OVER (
  PARTITION BY colonne_groupe   -- facultatif
  ORDER BY colonne_tri
  ROWS BETWEEN N PRECEDING AND CURRENT ROW  -- fenêtre glissante
)
```

Fonctions utiles :
| Fonction | Description |
|---|---|
| `LAG(col, n)` | Valeur de la colonne N lignes avant |
| `LEAD(col, n)` | Valeur de la colonne N lignes après |
| `AVG(col) OVER (...)` | Moyenne sur la fenêtre |
| `RANK() OVER (...)` | Rang dans la partition |
| `ROW_NUMBER() OVER (...)` | Numéro de ligne dans la partition |
| `SUM(col) OVER (...)` | Somme cumulative |

## Rappels techniques

```scala
// Importer les fonctions de fenêtre
import org.apache.spark.sql.expressions.Window
import org.apache.spark.sql.functions._

// Définir une fenêtre ordonnée par date
val windowByDate = Window.orderBy("Date")

// Appliquer LAG dans le DataFrame API
df.withColumn("close_veille", lag(col("Close"), 1).over(windowByDate))

// En SQL pur (recommandé pour ce TP) :
spark.sql("""
  SELECT Date, Close,
         LAG(Close, 1) OVER (ORDER BY Date ASC) AS close_veille
  FROM aapl
""")

// Fenêtre glissante de 7 jours
spark.sql("""
  SELECT Date, Close,
         AVG(Close) OVER (
           ORDER BY Date ASC
           ROWS BETWEEN 6 PRECEDING AND CURRENT ROW
         ) AS ma7
  FROM aapl
""")
```
